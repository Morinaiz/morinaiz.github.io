Pisa Population Density Map

Personal Project

Project Overview

In this personal project of mine, I created a heatmap to visually represent the population density of Pisa, my home city. This project was driven by my interest in understanding the demographic patterns of my city. To achieve this, I utilized high-resolution population density maps provided by Meta's Data for Good initiative, which I then extracted, processed, and displayed using Leaflet.

Technical Achievements

  • Data Extraction with Python:
    • Used Pandas for data processing, handling and manipulation.
    • Utilized Shapely for geographical data processing.
  • Visualization with Leaflet:
    • Implemented Leaflet for map visualization.
    • Created an interactive heatmap overlaying population data on a map of Pisa.
    • Ensured the heatmap was user-friendly and visually informative.
