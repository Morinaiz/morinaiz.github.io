Double Face Shader

Upwork Project

Project Overview

For an Upwork client, I successfully debugged and optimized a Unity shader to ensure uniform performance across WebGL, desktop, and mobile platforms. Initially displaying visual artifacts on mobile, the shader was reconstructed from zero, to maintain high performance and consistency across all platforms. This project was completed efficiently within the set timeline and budget.

Technical Achievements

  • Reconstructed shader logic for improved cross-platform compatibility.
  • Optimized performance and resource usage for mobile devices.
