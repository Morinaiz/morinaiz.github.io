Unicode Procedural Map Log

Personal Project

Project Overview

While developing my latest game, a roguelike real-time tactics (RTT) where players command platoons of cats, I implemented a Unicode-based log system to streamline debugging for the procedurally generated levels.

To maintain transparency in the game's development process, I decided to create a script that automatically generates a detailed Unicode breakdown of the map generation process every time I create a new map inside of Unity.

This map log is then uploaded to my website via GitHub Pages, using a secondary account.

You can check out the repository for the automated map logs here: lorenzogamedevautomatizedmaplogs.github.io

Below, you can see the Unicode Log of the last generated map. As time goes on, logs might change style, and I will add more details and stats.
