Cartoon Shader

Personal Project

Project Overview

A robust Cartoon Shader for Unity, crafted for my 3D RTS with cats. This is a classic Cel Shader with Cartoon aesthetic, delivering rich, vibrant visuals while being highly optimized.

Artistic Expression

The pixel-perfect outline detection accentuates object silhouettes, allowing me to create bold visuals, while the customization options allow fine-tuning of the colors, regardless of the lighting conditions. It supports up to 5 colors per Material, textures, Clouds, Day-Night Cycle, Multiple colored point lights, Outlines and much more!

Optimization

The shader delivers a high-definition cartoon effect, featuring a palette of rich colors and crisp transitions. It allows for up to 5 colors per Material, and is designed for optimal performance. It runs smoothly across various hardware configurations, including mobile, while minimizing resource-intensive operations.

Technical Achievements

  • Outline: Dynamic edge detection using normal and depth data for adjustable outline thickness.
  • Texture and Normal: Integration of normal maps ensures the preservation of texture detail within a cartoon aesthetic.
  • Clouds: Efficient rendering of real-time cloud shadows for performance optimization.
  • Fresnel: Realistic implementation of the Fresnel effect for enhanced multi-light interaction on surfaces.
  • Performance: Optimized shader performance across multiple platforms, including mobile devices.
  • 5 Colors Palette: A customizable five-color palette for greater artistic flexibility and control.
